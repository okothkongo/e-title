# ETitle - Digital Land Records Verification System

## Introduction
ETitle is a secure centralized digital platform aimed at modernizing Kenya's land registry system. The project addresses long-standing inefficiencies in land ownership and management by digitizing records, automating verification, and increasing transparency in land transactions.

## Key Features
- Secure digital land title records with blockchain-based immutable audit trails
- Real-time verification of land ownership status
- Centralized database unifying land records across counties  
- Role-based access for administrators, officials, citizens and stakeholders
- User-friendly web interface for easy access

## Problem Statement
Kenya's current manual, paper-based land titling system faces numerous challenges:
- Human errors and processing delays
- Vulnerability to fraud and corruption
- Physical degradation of critical documents
- Fragmented records across different locations
- Limited accessibility for citizens in remote areas

## Project Scope
The system focuses on:
- Digitizing and securing land titles/ownership records
- Providing real-time ownership verification
- Enabling role-based access control
- Creating immutable audit trails via blockchain
- Streamlining land transactions

## Project Impact
- Estimated to save Kenya Ksh 100 billion in annual legal fees and lost productivity
- Expected to reduce title issuance time by up to 80%
- Aligns with Kenya's Vision 2030 goals for modernizing governance
- Sets precedent for digital transformation in public services

## Getting Started

### Development Setup
1. Run `mix setup` to install dependencies
2. Start Phoenix server:
   - `mix phx.server` or
   - `iex -S mix phx.server` (interactive)
3. Visit [`localhost:4000`](http://localhost:4000) in your browser

### Production Deployment
For production deployment instructions, see the [Phoenix deployment guides](https://hexdocs.pm/phoenix/deployment.html).

## Limitations
The system focuses solely on technological solutions for digital record management and verification. It does not address:
- Physical land surveying
- Historical land dispute resolution  
- Legal reforms
- Institutional capacity gaps

These aspects require separate policy and legislative interventions.
